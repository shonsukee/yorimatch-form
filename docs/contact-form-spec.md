# Yorimatch お問い合わせフォーム仕様書

## 概要

本フォームは `Yorimatch` 利用者向けの簡易サポート窓口である。静的ホスティング環境での運用を前提とし、送信ボタン押下で `yorimatch.support@gmail.com` 宛てのメール作成画面を開く。

## 画面構成

- 固定ヘッダー
  - タイトル: `Yorimatch サポート`
- 本文案内
  - 問い合わせ受付の説明
  - 送信ボタンでメールアプリが起動する旨
- 入力フォーム
  - お問い合わせ種別
  - 返信用メールアドレス
  - iOS バージョン
  - アプリバージョン
  - お問い合わせ内容
- エラーメッセージ
  - 必須項目未入力時に表示
- 送信ボタン
  - メールアプリを起動して送信する

## 入力項目

### 必須

- お問い合わせ種別
  - `バグ・不具合の報告`
  - `機能の要望`
  - `アプリの使い方について`
  - `その他`
- 返信用メールアドレス
- iOS バージョン
- アプリバージョン
- お問い合わせ内容

## 送信仕様

- 宛先: `yorimatch.support@gmail.com`
- 件名:

```text
[Yorimatch お問い合わせ] {お問い合わせ種別}
```

- 本文:

```text
※このまま送信してください。

【お問い合わせ種別】
...

【返信用メールアドレス】
...

【ご利用環境】
iOSバージョン: ...
アプリバージョン: ...

【お問い合わせ内容】
...
```

## バリデーション

- 必須項目が 1 つでも空の場合は送信しない
- エラーメッセージ `未入力または入力形式に誤りがある項目があります。` を表示する
- メールアドレスは `input[type="email"]` によるブラウザ標準バリデーションを利用する

## 実装方針

- `index.html` の単一ファイル構成
- Tailwind CDN を利用
- ダークモードは OS 設定に追従
- `mailto:` によるメール起動
